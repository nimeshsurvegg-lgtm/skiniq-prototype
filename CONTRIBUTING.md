# Contributing to SkinIQ

Thank you for your interest in improving SkinIQ!

## How Can I Contribute?

### 1. Reporting Bugs
If you encounter a bug, such as a UI glitch in the scan flow or an issue with the analysis logic, please open a new issue on GitHub. Please include:
* Your browser and device type.
* Steps to reproduce the issue.
* A description of the expected versus actual behavior.

### 2. Submitting Improvements
Since this is a single-file prototype, contributions that optimize the vanilla JavaScript logic or enhance the CSS-based themes are highly welcome.

1. **Fork** the repository.
2. **Clone** your fork locally.
3. **Create a branch** for your feature: `git checkout -b feature/NewFeature`
4. **Make your changes**. Please ensure the code remains compatible with standard modern browsers.
5. **Commit your changes**: `git commit -m "Add NewFeature"`
6. **Push** to your fork: `git push origin feature/NewFeature`
7. **Open a Pull Request** against the main branch.
