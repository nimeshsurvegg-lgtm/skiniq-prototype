## 🚀 Features

* **Multi-Angle AI Scanning:** Users capture front, left, and right profiles to map structural features, sebum levels, and pigmentation.
* **Intelligent Scoring:** Generates a Skin Health Score (0-100) and estimates skin age based on analysis.
* **Personalized Logic Engine:** The app combines scan data with questionnaire responses regarding diet, sun exposure, and lifestyle to suggest targeted treatments.
* **Dynamic Routine Builder:** Provides specific ingredient recommendations (e.g., Niacinamide, Salicylic Acid, Ceramides) based on the user's analyzed skin concerns.
* **User Authentication & History:** Securely saves scan history and routine progress for registered users, with a Guest mode for quick, anonymous analysis[cite: 3].
* **Premium UI/UX:** Features a responsive, modern interface with Light/Dark mode support[cite: 3].

## 🧠 Technical Architecture
**Frontend:** A high-performance, single-file prototype using vanilla HTML5, CSS3, and JavaScript[cite: 3].

**State Management:** Uses a local state object to handle authentication, multi-step capture flows, and form data[cite: 3].

**Persistence:** Leverages localStorage for user sessions, historical scan data, and theme preferences[cite: 3].

**No Dependencies:** Runs entirely in the browser; does not require external build tools or server-side frameworks[cite: 3].
